# aoc-2023

[English](README.md) | Français

Ce dépôt contient les solutions pour l'Advent of Code 2023. 

## Structure du Projet

Le projet est organisé avec une solution pour chaque jour du défi Advent of Code.

## Installation

Pour exécuter les solutions, assurez-vous d'avoir installé les dépendances nécessaires :

```bash
npm install
```

## Utilisation

Chaque solution peut être exécutée individuellement. Naviguez dans le répertoire du jour correspondant et exécutez la solution avec Node.js.
